# Company Performance Dashboard

## Project Description
- An interactive Power BI dashboard visualizing company performance.
- Analyze monthly sales, department-wise sales, and employee performance trends.
- Designed to help business users make informed decisions.

## Tools Used
- Power BI Desktop
- Python (for data preprocessing)
- Excel / CSV for dataset

## Dataset
- Columns: employee_id, employee_name, department, designation, location, month, monthly_salary, sales_amount, performance_rating
- Sample data for 20 employees over multiple months

## Dashboard Visuals
- **Monthly Sales Trend** – Line / Column chart
- **Department-Wise Sales Performance** – Bar chart
- **Relationship Between Sales and Performance Rating** – Scatter chart
- **Sales Contribution by Department** – Pie / Donut chart
- **Distribution of Employee Performance Ratings** – Column chart

## Slicers / Filters
- **Month** – filter visuals by month
- **Department** – compare departments
- **Performance Rating** – focus on high or low performers
- Optional: Location / Designation slicers

## Insights
- Sales department generates highest revenue
- Senior employees outperform juniors
- Performance rating correlates with sales trends
- Training and incentives can improve productivity

## Business Recommendations
- Reward top performers
- Train low-performing staff
- Align marketing & sales strategy
- Use performance ratings for promotion & incentives
